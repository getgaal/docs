[Continue](https://continue.dev) is an open-source coding assistant. Skills live under `.continue/skills`; MCP config lives under `~/.continue/config.json`.

## Paths

| Resource | Path |
|----------|------|
| Project skills | `.continue/skills` |
| Global skills | `~/.continue/skills` |
| MCP config | `~/.continue/config.json` |
| Audit search (project) | `.continue/skills`, `.agents/skills`, `.github/skills`, `.claude/skills` |
| Audit search (global) | `~/.continue/skills`, `~/.agents/skills`, `~/.copilot/skills` |

## Skills example

```yaml gaal.yaml
skills:
  - source: anthropics/skills
    agents: [continue]
    global: true
```

## MCP example

```yaml gaal.yaml
mcps:
  - name: filesystem
    agents: [continue]
    global: true
    inline:
      command: uvx
      args: [mcp-server-filesystem, ~/projects]
```
