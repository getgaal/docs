[Codex](https://github.com/openai/codex) is OpenAI's terminal coding agent. Project skills use the vendor-neutral `.agents/skills` convention; global skills live under `~/.codex/skills`.

## Paths

| Resource | Path |
|----------|------|
| Project skills | `.agents/skills` (generic fallback) |
| Global skills | `~/.codex/skills` |
| MCP config | `~/.codex/mcp.json` |
| Audit search (project) | `.agents/skills`, `.github/skills`, `.claude/skills` |
| Audit search (global) | `~/.codex/skills`, `~/.agents/skills`, `~/.copilot/skills` |

## Skills example

```yaml gaal.yaml
skills:
  - source: anthropics/skills
    agents: [codex]
    global: true       # → ~/.codex/skills
```

## MCP example

```yaml gaal.yaml
mcps:
  - name: filesystem
    agents: [codex]
    global: true
    inline:
      command: uvx
      args: [mcp-server-filesystem, ~/projects]
```
