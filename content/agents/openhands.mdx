[OpenHands](https://github.com/All-Hands-AI/OpenHands) is an open-source autonomous coding agent. Skills live under `.openhands/skills`; MCP config lives under `~/.openhands/config.json`.

## Paths

| Resource | Path |
|----------|------|
| Project skills | `.openhands/skills` |
| Global skills | `~/.openhands/skills` |
| MCP config | `~/.openhands/config.json` |
| Audit search (project) | `.openhands/skills`, `.agents/skills`, `.github/skills` |
| Audit search (global) | `~/.openhands/skills`, `~/.agents/skills` |

## Skills example

```yaml gaal.yaml
skills:
  - source: anthropics/skills
    agents: [openhands]
    global: false
```

## MCP example

```yaml gaal.yaml
mcps:
  - name: filesystem
    agents: [openhands]
    global: true
    inline:
      command: uvx
      args: [mcp-server-filesystem, ~/projects]
```
