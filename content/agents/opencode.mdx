[OpenCode](https://opencode.ai) is an open-source terminal coding agent. Project skills use the vendor-neutral `.agents/skills` convention; global skills live under `~/.config/opencode/skills`.

## Paths

| Resource | Path |
|----------|------|
| Project skills | `.agents/skills` (generic fallback) |
| Global skills | `~/.config/opencode/skills` |
| MCP config | `~/.config/opencode/config.json` |
| Audit search (project) | `.agents/skills`, `.github/skills`, `.claude/skills` |
| Audit search (global) | `~/.config/opencode/skills`, `~/.agents/skills`, `~/.copilot/skills` |

## Skills example

```yaml gaal.yaml
skills:
  - source: anthropics/skills
    agents: [opencode]
    global: true
```

## MCP example

```yaml gaal.yaml
mcps:
  - name: filesystem
    agents: [opencode]
    global: true
    inline:
      command: uvx
      args: [mcp-server-filesystem, ~/projects]
```
