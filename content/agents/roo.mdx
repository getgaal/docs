[Roo Code](https://roocode.com) is an AI assistant for VS Code. Skills live under `.roo/skills`; MCP config lives in VS Code's `settings.json`.

## Paths

| Resource | Path |
|----------|------|
| Project skills | `.roo/skills` |
| Global skills | `~/.roo/skills` |
| MCP config | `~/.vscode/settings.json` |
| Audit search (project) | `.roo/skills`, `.agents/skills`, `.github/skills`, `.claude/skills` |
| Audit search (global) | `~/.roo/skills`, `~/.agents/skills`, `~/.copilot/skills` |

## Skills example

```yaml gaal.yaml
skills:
  - source: anthropics/skills
    agents: [roo]
    global: false
```

## MCP example

```yaml gaal.yaml
mcps:
  - name: filesystem
    target: ~/.vscode/settings.json
    inline:
      command: uvx
      args: [mcp-server-filesystem, ~/projects]
```
